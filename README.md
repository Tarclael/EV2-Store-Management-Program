	2. Search by ID
	3. Add product
	4. Modify product
		1. Change name
		2. Change price
		3. Change quantity
		4. Change supplier
		5. Change inventory

	6. Return to Main Menu

	1. See all inventory

	3. Search by ID
	4. Add inventory

	6. Remove inventory

3. Manage supplier
	1. See all supplier
	2. Search by ID
	3. Add supplier
	4. Modify supplier
		1. Change name
		2. Change email
		3. Change contact number
		4. Change address
		5. Back
	5. Remove supplier
	6. Return to Main Menu
4. Manage employee
	1. See all employees
	2. Search employee by ID
	3. Add employee
	4. Modify employee
		1. Change name
		2. Change role
		3. Change salary
		4. Back
	5. Remove employee
	6. Give bonus
	7. Return to Main Menu
